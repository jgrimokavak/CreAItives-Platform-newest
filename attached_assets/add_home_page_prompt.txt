====================================

Objective:

   - **File**: `pages/HomePage.tsx`
     ```tsx
     export default function HomePage() {

2. Register a route for Home

       // existing items continue…
       ```
     - Adjust active styling logic to work with root path.

4. Active route highlighting
   - Keep the current `NavLink` logic; verify the `to="/"` link gets `aria-current="page"` when on Home.

5. Responsive tweaks
   - If the sidebar collapses to icons on mobile, use `LucideHome` for consistency.

6. Update default redirect after auth
   - In any `PrivateRoute` or auth logic, change post‑login redirect from `/create` to `/`.

Reminders & Constraints
- Follow existing Tailwind + shadcn UI classes (`rounded-lg px-3 py-2 hover:bg-accent/50` etc.).
- Keep consistent icon size (16 px) and text alignment.
